Plugins

This page lists available Hudson plugins (How do I host mine?)

Hudson Japex Plugin
This plugin adds Japex support for Hudson so that Hudson can display trend reports and other useful metrics.
java.net uploader plugin
This plugin uses java.net tasks library to make Hudson capable of posting artifacts to java.net

How to install?

Simply put ***.hpi file to $HUDSON_HOME/plugins directory. Then you need to restart Hudson (many containers let you do this without restarting the container.)
