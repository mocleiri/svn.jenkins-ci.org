Hudson RPM for RedHat Linux

Hudson can be installed as an RPM package, which enables:

  • Automatic upgrade of Hudson via yum
  • /etc/init.d hook up to start Hudson on boot
To use the repository, first add the key to your system: 
sudo rpm --import http://hudson-ci.org/redhat/hudson-ci.org.key

Then install Hudson: 

wget -O /tmp/hudson.rpm http://hudson-ci.org/latest/redhat/hudson.rpm
sudo rpm --install /tmp/hudson.rpm

Once installed, Hudson can be updated with yum.
