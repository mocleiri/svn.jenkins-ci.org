setSystemProperties

public void setSystemProperties(java.util.Properties props)
Sets the properties that the embedded Maven sees as system properties.

In various places inside Maven, System.getProperties() are still referenced, and still in other places, the values given to this method is only used as overrides and not the replacement of the real System.getProperties(). So Maven still doesn't quite behave as it should, but at least this allows Hudson to add system properties to Maven without really messing up our current JVM.
