public interface ExtensionPoint

Marker interface that designates extensible components in Hudson that can be implemented by plugins.

See respective interfaces/classes for more about how to register custom implementations to Hudson. See Extension for how to have Hudson auto-discover your implementations.

This interface is used for auto-generating documentation.
