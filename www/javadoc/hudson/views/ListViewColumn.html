extends java.lang.Object
implements ExtensionPoint, Describable<ListViewColumn>

Extension point for adding a column to ListView.

This object must have the column.jelly. This view is called for each cell of this column. The Item object is passed in the "job" variable. The view should render the <td> tag.

This object may have an additional columHeader.jelly. The default ColmnHeader will render ColumnCaption.

For now, ListView doesn't allow ListViewColumns to be configured (instead it just shows all the columns available in all()), but the intention is eventually make each ListViewColumn fully configurable like Publisher.
