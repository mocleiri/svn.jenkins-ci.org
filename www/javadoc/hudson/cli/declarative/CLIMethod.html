hudson.cli.declarative
Annotation Type CLIMethod

@Indexed
@Retention(value=RUNTIME)
@Target(value=METHOD)
@Documented
public @interface CLIMethod

Annotates methods on model objects to expose them as CLI commands.

You need to have Messages.properties in the same package with the CLI.command-name.shortDescription key to describe the command. This is used for the same purpose as CLICommand.getShortDescription().

If you put a CLIMethod on an instance method (as opposed to a static method), you need a corresponding CLI resolver method.

A CLI method can have its parameters annotated with ListBoxModel.Option and Argument, to receive parameter/argument injections.

A CLI method needs to be public.
