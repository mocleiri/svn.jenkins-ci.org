compare

public int compare(Queue.BuildableItem lhs,
                   Queue.BuildableItem rhs)
Override this method to provide the ordering of the sort.

if lhs should be build before rhs, return a negative value. Or put another way, think of the comparison as a process of converting a Queue.BuildableItem into a number, then doing num(lhs)-num(rhs).

The default implementation does FIFO.
