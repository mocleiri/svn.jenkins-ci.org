Class UpdateSite

java.lang.Object
  hudson.model.UpdateSite

public class UpdateSite
extends Object

Source of the update center information, like "http://hudson-ci.org/update-center.json"

Hudson can have multiple UpdateSites registered in the system, so that it can pick up plugins from different locations.

Since:
1.333
Author:
