public static class UpdateCenter.UpdateCenterConfiguration
extends java.lang.Object
implements ExtensionPoint

Configuration data for controlling the update center's behaviors. The update center's defaults will check internet connectivity by trying to connect to www.google.com; will download plugins, the plugin catalog and updates from hudson.dev.java.net; and will install plugins with file system operations.
