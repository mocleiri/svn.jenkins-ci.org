public static class Queue.JobOffer
extends Object

Data structure created for each idle Executor. This is a job offer from the queue to an executor.

An idle executor (that calls Queue.pop() creates a new Queue.JobOffer and gets itself parked, and we'll eventually hand out an item to build.
