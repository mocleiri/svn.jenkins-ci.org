public class LogRecorder
extends AbstractModelObject
implements Saveable

Records a selected set of logs so that the system administrator can diagnose a specific aspect of the system. TODO: still a work in progress.

Access Control

LogRecorder is only visible for administrators, and this access control happens at Hudson.getLog(), the sole entry point for binding LogRecorder to URL.
