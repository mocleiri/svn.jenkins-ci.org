hudson:replace-job

Full name:

com.mtvi.plateng.maven:maven-hudson-plugin:2.0-alpha-1:replace-job

Description:

Goal which replaces a Hudson job's configuration

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Executes as an aggregator plugin.

Required Parameters

Name Type Since Description
includeCobertura boolean - Should the produced project XML include Cobertura Publisher support. Default value is: true.
includeEmail boolean - Should the produced project XML include email support. Default value is: true.
includeJavadoc boolean - Should the produced project XML include Javadoc Publisher support. Default value is: true.
includeJunit boolean - Should the produced project XML include JUnit Publisher support. Default value is: true.
includeTwitter boolean - Should the produced project XML include twitter support. Default value is: true.
includeViolations boolean - Should the produced project XML include Violations Publisher support. Default value is: true.
primaryGoal String - Primary "build" goal to be executed. Usually either intstall or deploy. Default value is: install.
subversionURL String - Subversion URL.

Optional Parameters

Name Type Since Description
additionalGoals String - Additional goals to be run after any reporting goals are executed.
allocatePorts String - Comma-delimited list of environment variables to have set using the Port Allocator plugin. See http://hudson.gotdns.com/wiki/display/HUDSON/Port+Allocator+Plugin
description String - Project description.
hudsonURL URL - Hudson URL.
jobName String - Name of the job to be created.
performClean boolean - Should a clean be performed before the build. Default value is: true.
startVNC boolean - If true, will start a VNC session before each job execution. See http://hudson.gotdns.com/wiki/display/HUDSON/Xvnc+Plugin.
