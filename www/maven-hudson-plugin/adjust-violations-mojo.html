hudson:adjust-violations

Full name:

org.jvnet.hudson.tools:maven-hudson-plugin:2.0-alpha-2:adjust-violations

Description:

Mojo that changes the settings of the Violations Hudson plugin.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
max int - Maximum number of violations for Cloudy status.
min int - Minimum number of violations for Cloudy status.
unstable int - Violation number at which the build becomes unstable.

Optional Parameters

Name Type Since Description
hudsonURL URL - Hudson URL.
jobName String - The job name.

Parameter Details

hudsonURL:

Hudson URL.

  • Type: java.net.URL
  • Required: No
  • Expression: ${hudson.url}

jobName:

The job name.

  • Type: java.lang.String
  • Required: No
  • Expression: ${jobName}

max:

Maximum number of violations for Cloudy status.

  • Type: int
  • Required: Yes
  • Expression: ${max}

min:

Minimum number of violations for Cloudy status.

  • Type: int
  • Required: Yes
  • Expression: ${min}

unstable:

Violation number at which the build becomes unstable.

  • Type: int
  • Required: Yes
  • Expression: ${unstable}
