hudson:add-job

Full name:

org.jvnet.hudson.tools:maven-hudson-plugin:2.0-alpha-4:add-job

Description:

Goal which creates a Hudson job.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Executes as an aggregator plugin.

Required Parameters

Name Type Since Description
includeCobertura boolean - Should the produced project XML include Cobertura Publisher support.
Default value is: true.
includeEmail boolean - Should the produced project XML include email support.
Default value is: true.
includeJavadoc boolean - Should the produced project XML include Javadoc Publisher support.
Default value is: true.
includeJunit boolean - Should the produced project XML include JUnit Publisher support.
Default value is: true.
includeTwitter boolean - Should the produced project XML include twitter support.
Default value is: true.
includeViolations boolean - Should the produced project XML include Violations Publisher support.
Default value is: true.
primaryGoal String - Primary "build" goal to be executed. Usually either intstall or deploy.
Default value is: install.
scmUrl String - SCM URL.

Optional Parameters

Name Type Since Description
additionalGoals String - Additional goals to be run after any reporting goals are executed.
allocatePorts String - Comma-delimited list of environment variables to have set using the Port Allocator plugin. See http://hudson.gotdns.com/wiki/display/HUDSON/Port+Allocator+Plugin
description String - Project description.
hudsonURL URL - Hudson URL.
jobName String - Name of the job to be created.
performClean boolean - Should a clean be performed before the build.
Default value is: true.
shouldFailAtEnd boolean - Should the -fae command-line option be passed to Maven, specifying that the build should only report a failure at the end.
Default value is: true.
shouldUseBatchMode boolean - Should the -B command-line option be passed to Maven, specifying that the build is running in batch (i.e. non-interactive) mode.
Default value is: true.
startVNC boolean - If true, will start a VNC session before each job execution. See http://hudson.gotdns.com/wiki/display/HUDSON/Xvnc+Plugin.
useMavenProjectType boolean - True to use the Maven 2 Project Type
Default value is: false.
