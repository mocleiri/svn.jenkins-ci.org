hudson:add-twitter

Full name:

org.jvnet.hudson.tools:maven-hudson-plugin:2.0-alpha-4:add-twitter

Description:

Goal which adds Twitter notifications to a job or all jobs.

Attributes:

    Optional Parameters

    Name Type Since Description
    doAll boolean - Do the thing for every job?
    hudsonURL URL - Hudson URL.
    jobName String - The job name.

    Parameter Details

    doAll:

    Do the thing for every job?
    • Type: boolean
    • Required: No
    • Expression: ${doAll}

    hudsonURL:

    Hudson URL.
    • Type: java.net.URL
    • Required: No
    • Expression: ${hudson.url}

    jobName:

    The job name.
    • Type: java.lang.String
    • Required: No
    • Expression: ${jobName}
