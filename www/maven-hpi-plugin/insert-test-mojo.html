hpi:insert-test

Full name:

org.jvnet.hudson.tools:maven-hpi-plugin:1.45:insert-test

Description:

Insert default test suite.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: generate-test-sources.
