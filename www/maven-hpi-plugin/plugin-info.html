Plugin documentation

Goals available:

GoalDescription
hpi:announcePosts a release announcement.
hpi:apt-compileNo description
hpi:createBuilds a new plugin template.

Most of this is really just a rip-off from the archetype:create goal, but since Maven doesn't really let one Mojo calls another Mojo, this turns out to be the easiest.

hpi:hpiBuild a war/webapp.
hpi:hplGenerate .hpl file.
hpi:runRuns Hudson with the current plugin project.

This only needs the source files to be compiled, so run in the compile phase.

To specify the HTTP port, use -Djetty.port=PORT

hpi:uploadPosts the hpi file to java.net.
