Maven Hudson Plugin

This maven2 plugin provides various goals for developing Hudson plugins.

Goals Overview

  • hpi:create Creates a skeleton of a new plugin.
  • hpi:hpi Builds the .hpi file
  • hpi:hpl Generates the .hpl file
  • hpi:run Runs Hudson with the current plugin project
  • hpi:upload Posts the hpi file to java.net. Used during the release.

    For more information about how to develop Hudson plugins, refer to Wiki
