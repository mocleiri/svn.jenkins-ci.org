Dieses Plugin ermöglicht die Authentifizierung über PAM mit Hilfe des Tools pwauth, welches üblicherweise mit Apache und dem mod_authnz_external Modul verwendet wird.

Das Plugin unterstützt Gruppen und liest alle Einstellungen direkt aus dem unterliegenden *nix System.
WINDOWS WIRD NICHT UNTERSTÜTZT!

Für weitere Informationen, wie man pwauth einrichtet besucht man am besten deren Projektseite.
