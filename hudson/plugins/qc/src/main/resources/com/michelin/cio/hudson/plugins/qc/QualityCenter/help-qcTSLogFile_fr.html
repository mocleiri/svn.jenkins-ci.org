Nom du rapport à générer.

Si plusieurs TestSets doivent être exécutés, alors un rapport sera généré par TestSet. En conséquence, vous devez vous assurer que le nom de chaque rapport sera unique en y incluant le nom du TestSet correspondant (grâce à la variable ${TS_NAME} variable). Si ça n'est pas le cas, alors le nom sera automatiquement modifié : il sera suffixé par un souligné suivi du nom du TestSet.

Vous pouvez utiliser toutes les variables d'environnement usuelles plus :

  • ${QC_DOMAIN} pour le domaine Quality Center ;
  • ${QC_PROJECT} pour le projet Quality Center ;
  • ${TS_FOLDER} pour le répertoire du TestSet ;
  • ${TS_NAME} pour le nom du TestSet.
