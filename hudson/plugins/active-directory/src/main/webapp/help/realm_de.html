Wenn diese Option aktiviert ist, authentifiziert Hudson die angegebenen Benutzernamen und Passwörter gegen das Active Directory.

Hudson erkennt alle Gruppen im Active Directory, in denen der Benutzer Mitglied ist. Daher können auch Gruppenzugehörigkeiten für die Authorisierung eingesetzt werden. Beispielsweise kann mit der Authorisierungsstrategie "Matrix-basierte Sicherheit" jedem 'Domänen Administrator' die Administration von Hudson erlaubt werden.
