Filename of a 16x16 badge icon to show in the Build History. Enter a path starting with /userContent for a custom image in the $HUDSON_HOME/userContent directory (such as /userContent/mybadge.gif). Enter a plain filename for a file in either $HUDSON_HOME/plugins/promoted-builds-simple/images (plugin includes ga.gif, qa.gif and qa-green.gif) or in Hudson's images/16x16 directory (such as star-gold.gif).
