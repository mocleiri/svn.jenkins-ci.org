Les libellés (ou tags) sont utilisés pour grouper plusieurs esclaves dans un groupe logique.

Par exemple, si vous avez de multiples esclaves Windows et des jobs qui nécessitent Windows, configurez vos esclaces Windows avec un tag 'windows', puis rattachez les jobs au tag 'windows'. Cela pemettra à vos jobs de ne s'exécuter que sur les esclaves Windows et non sur les autres.
