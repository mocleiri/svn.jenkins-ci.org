Controls how Hudson permits launching via JNLP.
Launch only from Computer detail. Require Login if security enabled.
This is the default and normal setting. In this mode, Hudson provides the JNLP launch link on the computer view. If Security is enabled, then the link will only be available to authenticated users.
Launch from front page. Require Login if security enabled.
In this mode, Hudson provides the JNLP launch link on both the computer view and the Build Executor Status sidepanel. If Security is enabled, then the link will only be available to authenticated users.
Launch only from Computer detail. Login never required.
In this mode, Hudson provides the JNLP launch link on the computer view. The link will always be available whenever the slave is offline.
Launch from front page. Login never required.
In this mode, Hudson provides the JNLP launch link on both the computer view and the Build Executor Status sidepanel. The link will always be available whenever the slave is offline.
