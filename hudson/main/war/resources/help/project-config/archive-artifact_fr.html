Archive les artefacts du build (par exemple, les fichiers zip de la distribution ou les fichiers jar) afin permettre leur téléchargement. Les fichiers archivés sont accessibles à partir de la page Hudson du build.
En général, Hudson conserve les artefacts d'un build aussi longtemps que le log du build. Néanmoins, si vous n'avez pas besoin des artefacts anciens et préférez gagner de l'espace disque, il est possible de les supprimer.
