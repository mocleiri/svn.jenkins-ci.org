Projeyi yapılandırmak için shell scripti (varsayılan sh'dır, fakat değiştirilebilir) çalıştırır. Script, bulunduğu dizin, çalışma alanı olacak şekilde çalışır. Shell "-ex" seçeneği ile çağırılır. Böylece tüm komutlar çalışmadan önce ekrana yazılır ve komutlardan herhangi birisi 0 olmayan bir çıkış kodu ile biterse yapılandırma başarısız olarak kabul edilir.
