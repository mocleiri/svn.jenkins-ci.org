Lance un script shell (par défaut avec sh, mais cela est configurable) pour construire le projet. Le script sera lancé avec le workspace comme répertoire courant. Le shell sera invoqué avec l'option "-ex". Par conséquent, toutes les commandes seront affichées avant d'être exécutées, et le build sera considéré comme un échec si une de ces commandes renvoie un code de retour non-nul.
