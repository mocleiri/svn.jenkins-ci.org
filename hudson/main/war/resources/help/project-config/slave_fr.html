Parfois, un projet ne peut être construit que sur une machine-esclace particulière. Dans ce cas, cette option force Hudson à toujours construre ce projet sur une machine particulière. Dans le cas contraire, décochez la case afin que Hudson puisse programmer des builds sur les machines disponibles, ce qui permettra un meilleur temps de réponse.

Cette option est également utile quand vous souhaitez vous assurer qu'un projet peut être construit sur un noeud (une machine) particulier.
