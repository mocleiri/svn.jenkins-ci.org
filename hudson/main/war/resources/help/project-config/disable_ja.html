Sometimes, you'd want to temporarily stop building a project. For example, maybe you are in the middle of a large migration and you know new builds are going to fail. Or maybe a project is built every hour but you know that the CVS server will be down for the next 24 hours. 時々、プロジェクトのビルドを止めたいと思うことがあるでしょう。 例えば、大きなマイグレーションの途中であったり、新しいビルドが失敗すると知っているような場合です。 あるいは、プロジェクトは毎時間ビルドされますが、CVSサーバーが次の24時間ダウンしているだろうことを知っているような場合です。

このオプションをセットしたとき、新しいビルドはこのプロジェクト上では行われません。 ビルドの依存チェーンのうちのどれも変更せず、セット・アップされた通知を変更せずに、新しいビルドを不能にすることができます。
