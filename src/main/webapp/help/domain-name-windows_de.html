(Optional) Spezifiziert das Active Directory, gegen das authentifiziert werden soll.

Ohne Angabe nutzt Hudson ADSI um mit dem Active Directory Kontakt aufzunehmen. Dieses benötigt keinen konfigurierten Domänennamen.

Es wurden allerdings Probleme gemeldet, wenn Hudson unter einem Benutzerkonto läuft, das nicht Mitglied der Domäne ist. Dies ist häufig dann der Fall, wenn Hudson als Dienst installiert wird. In solchen Fällen, muss der Domänenname hier eingetragen werden. Hudson nutzt dann LDAP für die Kommunikation mit dem Active Directory.
